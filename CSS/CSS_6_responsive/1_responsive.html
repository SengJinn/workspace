반응형 웹이란?

  • 웹 사이트의 내용을 그대로 유지하면서 다양한 화면의 크기에 맞춰 웹사이트를 표시하는 방법
  • 화면의 크기에 반응해 화면 요소들을 자동으로 바꾸어 사이트를 구현하는 것
  • 가로 모드에 맞춘 레이아웃 변경 가능
  • 사이트 유지 보수가 용이

반응형 웹 구현을 위한 기술

  • 그리드 레이아웃
    • 배치된 요소를 여러 개의 행(row)과 열(column)로 나누어, 필요할 때마다 병합하여 배치하는 격자(grid) 표현 방법
    • column : 열
    • row : 행

  • @media 쿼리
    • 접속하는 장치(미디어) 또는 크기에 따라 특정한 css 스타일을 사용하도록 지정하는 쿼리
    • 기본형태 :
      @media [only | not] 미디어 유형 [and 조건] * [and 조건]{ 적용할 디자인 속성 }
    • 미디어 유형
      all: 모든 미디어 유형
      print: 인쇄 장치
      screen : 장치의 화면
      tv: 음성과 영상이 동시에 출력되는 미디어
      project: 프로젝트
